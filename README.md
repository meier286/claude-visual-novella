# Голос в трубке

**Информация об авторе:** Raccoons, Кибербезопасность, группа СДП-КБ-241  
**Команда проекта:** Зайцев Максим, Супрун Ярослав, Петровский Антон

**Краткое описание:**  
Интерактивная веб-новелла и тренажер для проверки бдительности пользователей при получении звонков от телефонных мошенников. Проект реализован на чистом HTML, CSS и JavaScript (без бэкенда). Игроку предстоит распознать приемы мошенников (искусственная срочность, давление авторитетом) и сделать правильные выборы, управляя скрытыми параметрами «Стресс» и «Бдительность». Включает конструктор собственных сценариев диалогов.

## 📋 Требования
* Любой современный браузер (Chrome, Firefox, Edge, Safari)
* Node.js 14+ и npm (для удобного локального запуска)

## 🚀 Как запустить
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/[ваш-логин]/[имя-репо].git
   cd [имя-репо]

2. Установите зависимости (локальный веб-сервер):
   ```bash
   npm install

3. Запустите проект:
   ```bash
   npm start

(Альтернативный способ без Node.js: просто откройте файл main.html двойным кликом в любом браузере).
